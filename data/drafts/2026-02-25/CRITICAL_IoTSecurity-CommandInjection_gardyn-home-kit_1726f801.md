
# Gardyn Home Kit

> Gardyn Home Kit 스마트 가드닝 시스템에서 발견된 다수의 취약점은 농업 및 식품 분야의 중요 인프라 보안에 중대한 위협을 제기합니다. CISA가 발표한 권고안에 따르면, 공격자는 인증 없이 에지 기기에 접근하여 제어권을 획득하고 클라우드 환경의 사용자 정보에 접근할 수 있는 심각한 결함이 확인되었습니다. 이러한 취약점은 특히 미국 내 광범위하게 배포된 스마트 홈 기기를 표적으로 합니다.

주요 기술적 분석 결과, CVE-2025-29631은 입력값 검증 부재로 인한 OS 커맨드 주입을 허용하며, CVE-2025-1242는 API 및 펌웨어 역공학을 통해 하드코딩된 관리자 자격 증명을 추출할 수 있게 합니다. 또한, Azure IoT Hub 연결 문자열이 평문 HTTP를 통해 전송되거나 취약한 기본 SSH 자격 증명이 사용되는 등 현대적 IoT 보안 요구사항을 충족하지 못하는 여러 설계 결함이 발견되었습니다.

영향 평가 측면에서, 공격자는 이러한 결함을 악용하여 사용자 네트워크 내부로 침투하거나 Gardyn 클라우드 환경 내의 다른 기기로 확산하는 피벗(Pivot) 지점으로 활용할 수 있습니다. 이는 개인 정보 유출은 물론, 식품 생산과 관련된 하위 시스템의 가용성에 심각한 타격을 줄 수 있습니다.

대응 방안으로 운영자는 즉시 펌웨어를 master.619 버전 이상으로 업데이트하고 모바일 애플리케이션을 2.11.0 이상으로 갱신해야 합니다. 또한, IoT 기기를 내부 비즈니스 네트워크와 물리적 또는 논리적으로 격리하고, 방화벽 및 VPN을 사용하여 인터넷 노출을 최소화하는 심층 방어(Defense-in-Depth) 전략을 수립해야 합니다.

**Severity**: CRITICAL (9.1/10)
**Tags**: IoT Security, Command Injection, Hard-coded Credentials, Critical Infrastructure, Cloud Security
